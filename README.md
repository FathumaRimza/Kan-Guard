# 🛡️ Kan-Guard

## Child Protection and Community Safety Management System

Kan-Guard is a web-based child protection platform developed using the MERN stack. It helps parents, schools, and police collaborate to protect children by managing safety reports, missing child cases, alerts, and anonymous support.

---

## Features

### Parent
- Register children
- Submit child safety reports
- View community alerts
- Report missing children
- Anonymous support chatbot
- Receive notifications

### Police
- Review safety reports
- Update report status
- Manage missing child cases
- Publish emergency alerts
- View dashboard analytics

### School Administrator
- Review reports
- Publish school alerts
- Monitor child safety
- Dashboard statistics

## Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=Your MongoDB URI
JWT_SECRET=Your JWT Secret

EMAIL_USER=Your Gmail
EMAIL_PASS=Your Gmail App Password
```
